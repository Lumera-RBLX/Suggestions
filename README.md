# Suggestions

This repository is the public suggestion board for **Lumera** products.

Use issues in this repository to propose:
- **New product ideas** Lumera can create
- **Updates or improvements** to existing Lumera products

## How to submit a suggestion

1. Open a new issue in this repository.
2. Choose the **Product Suggestion** template.
3. Provide:
   - Your suggestion
   - Whether it is a **new product** or an **update**
   - What the product (or update) should do
   - Any relevant reference material (optional)

Clear, detailed suggestions help Lumera evaluate and prioritize ideas faster.
